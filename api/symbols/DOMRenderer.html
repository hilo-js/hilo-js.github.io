DOMRenderer

继承 DOMRenderer Renderer 模块 hilo/renderer/DOMRenderer 依赖 hilo/core/Class, hilo/core/Hilo, hilo/renderer/Renderer 源码 DOMRenderer.js
DOM+CSS3渲染器。将可视对象以DOM元素方式渲染出来。
