方法详情

[Static] create(properties:Object):Object
此方法是在Hilo中创建类的主要方法。
参数
properties:Object — 要创建的类的相关属性和方法。主要有：
  • Extends - 指定要继承的父类。
  • Mixes - 指定要混入的成员集合对象。
  • Statics - 指定类的静态属性或方法。
返回值
Object — 创建好的类。
