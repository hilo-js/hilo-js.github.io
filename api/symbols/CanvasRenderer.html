CanvasRenderer


继承 CanvasRenderer Renderer 模块 hilo/renderer/CanvasRenderer 依赖 hilo/core/Class, hilo/renderer/Renderer 源码 CanvasRenderer.js
canvas画布渲染器。所有可视对象将渲染在canvas画布上。
