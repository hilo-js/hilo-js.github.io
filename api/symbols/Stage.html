Stage


继承 Stage Container View 模块 hilo/view/Stage 依赖 hilo/core/Hilo, hilo/core/Class, hilo/view/Container 源码 Stage.js
舞台是可视对象树的根，可视对象只有添加到舞台或其子对象后才会被渲染出来。
