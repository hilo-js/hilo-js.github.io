BitmapText


继承 BitmapText Container View 模块 hilo/view/BitmapText 依赖 hilo/core/Class, hilo/core/Hilo, hilo/view/Container 源码 BitmapText.js
BitmapText类提供使用位图文本的功能。当前仅支持单行文本。
