Flappy Bird

Flappy Bird 是某段时间IOS上红极一时的一个简单的像素设计小游戏。我们用Hilo做了简单的复刻，帮助你快速了解怎样用Hilo实现一个类似这样的小游戏。

Graphics Demo

Hilo 包装了一些常用的基础图形绘制api，这个demo展示了一些基础常用的图形绘制方法和结果。同时支持SVG path的解析。快来试试看！

Bitmap Demo

如果你要使用Hilo绘制一张静态的图片，或者图片上的某个区域，看demo是最直接的办法！

Sprite Demo

做游戏，我们最长用到，也是最重要的就是动画精灵，为了更好的控制和渲染精灵，Hilo提供了一个便于管理的纹理集，简单的demo可以看这里。

Dom 混合

Web Game并不是只要canvas就够了，适当的使用dom来辅助能更好的实现某些功能。Hilo提供了一套几乎无缝的api允许你添加dom的元素到舞台上，并同样易于控制。
